This is the page to understand the rules for naming variables

        Use camelCase.
        Choose meaningful names.
        Start names with a letter, _, or $.
        Avoid JavaScript keywords.
        Avoid spaces and special characters.




        //user namemust begin with this type of letter
        let name;
        let _count;
        let $price;

        let 1user; // Invalid: cannot start with a number


        Use camelCase for variable names
        let userName;
        let totalAmount;
        let isLoggedIn;



        Bad Examples
        let x = "John"; // Not descriptive
        let user_123_abc = 25; // Unclear naming
        let 1user = "Alice"; // Invalid syntax


        Good Examples
        let firstName = "John";
        let totalPrice = 100;
        let isAvailable = true;
